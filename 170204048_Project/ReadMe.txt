This folder acts as a complete repository of a machine learning project "Google Playstore Apps Rating Prediction".
The files here are:
    - Google-Playstore.csv file which is the dataset for our project
    - Google-Playstore_Documentation.pdf file which contains the documentation for our dataset
    - multiple_linear_regression.ipynb which contains the code for training our dataset using multiple linear regression
    - polynomial_regression.ipynb which contains the code for training our dataset using polynomial regression
    - 170204048_Report.pdf which contains:
                    - The description of the project
                    - A brief description of the dataset
                    - Description of Multiple Linear Regression and Polynomial Regression
                    - Comparison of the r2_score,MAE,MSE,RMSE of both models 
                    - Conclusions generated from model analysis
